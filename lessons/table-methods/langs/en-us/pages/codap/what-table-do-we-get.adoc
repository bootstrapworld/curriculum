|===

| `order-by-age`

Transformer: Sort

Formula expression: age

Direction: ascending

|*1*@ifsoln{-B} ||*A*
| Produces a table with Toggle, Fritz, and Nori - but not Sunflower.

| `order-by-weight`

Transformer: Sort

Formula expression: pounds

Direction: descending
|*2*@ifsoln{-F} ||*B*
| Produces a table of all four animals, sorted youngest-to-oldest

| `filter-is-old`

Transformer: Filter

|*3*@ifsoln{-C} ||*C*
