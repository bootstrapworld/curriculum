== What does the Code Do?
The questions below rely on the following contracts:

[%AUTOWIDTH, cols="<,<",frame="none"]
|===
|
@show{(contract "add-centroid" '(Table String List<String>) "Table")} +
@show{(contract "liked-ids" '(Table) "List<String>")} +
|
@show{(contract "disliked-ids" '(Table) "List<String>")} +
@show{(contract "tagged-ids" '(Table String) "List<String>")}
|===

@n What code will add a centroid called `"SNOW"` to the `decorated-bag` table, for image IDs "4" and "5"? +
@fitb{}{@ifsoln{`add-centroid(decorated-bag, "SNOW", [list: "4", "5"])}}


*The code below defines `liked` as a list of IDs for all liked images in `decorated`:*

@center{`liked = liked-ids(decorated-bag)`}

@n What code will define `disliked` as a list of IDs for disliked all images in `decorated`? +
@fitb{}{@ifsoln{`disliked = disliked-ids(decorated-bag)`}}

@n What code will define `sun-tagged` as a list of IDs for all images in `decorated` tagged with "sun"? +
@fitb{}{@ifsoln{`sun-tagged = tagged-ids(decorated-bag, "sun")`}}

@n What code will add a centroid for all liked images in `decorated`?
@fitb{}{@ifsoln{add-centroid(decorated-bag, "LIKED", liked)}}

@n What code will add a centroid for all images in `decorated` tagged with "sun"?
@fitb{}{@ifsoln{add-centroid(decorated-bag, "LIKED", sun-tagged)}}

@n What code will compute the `cosine-similarity` for all liked images, using only the columns `"green"` and `"red"`? +
@fitb{}{@ifsoln{`cosine-similarity(add-centroid(decorated-bag, "LIKED", sun-tagged), "LIKED CENTROID", [list: "green", "red"])`}}
