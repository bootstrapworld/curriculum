== Searching in Luminance-Entropy Space

@n Images 4 and 5 are both "snowy". Which images are "grassy"? @fitb{}{@ifsoln{Images 3 and 6}}

@n Evaluate `simple-similarity(decorated2, "4", [list: "luminance", "entropy"])`. Which image ID was the closest match to `"4"`? @fitb{}{@ifsoln{`"0"`}}

@n Try again, this time using `distance-similarity`. Which image ID was the closest match to `"4"`? @fitb{}{@ifsoln{`"3"`}}

@n Try again, this time using `angle-similarity`. Which image ID was the closest match to `"4"`? @fitb{}{@ifsoln{`"5"`}}

@n How well  `angle-similarity` work for image 3, using the same dimensions? @fitb{}{@ifsoln{not well!}}

@n Evaluate `image-scatter-plot(decorated2, "luminance", "entropy", doc-thumbnail)`. Can you explain why searching in Luminance-Entropy Space worked well for our snowy images, but not the grassy ones?
