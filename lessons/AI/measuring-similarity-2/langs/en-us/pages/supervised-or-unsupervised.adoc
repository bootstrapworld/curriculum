Machine learning models are all around us, using a mix of different learning methods. Can you tell the difference between @vocab{supervised learning} and @vocab{unsupervised learning}?

[.FillVerticalSpace, cols="1a, <2a", options="header"]
|===
| Example
| What Kind of Learning and Why? (Circle one)
@scrub{
| @n A streaming platform recommends new movies to a user based only on what they've seen before (not what they've liked or disliked), and does not use any labeled information about the genre, cast, etc.
| Supervised @hspace{3em} @ifsoln-choice{Unsupervised}



@blanklines{3}{
@ifsoln{The platform has a list of points (viewing history) and is blindly recommending other, nearby points. It has no idea what any of the points represent, and nothing has been labeled}
}
}
| @n An e-commerce website uses historical data of past transactions, which are explicitly marked as 'authorized' or 'fraudulent', to train a model that flags suspicious new transactions.
| @ifsoln-choice{Supervised} @hspace{3em} Unsupervised

@blanklines{3}{
@ifsoln{This is a classification task, similar to what we saw in the Decision Trees lessons! The model is trained on a labeled dataset where the correct output ('authorized' or 'fraudulent') is known, making it a supervised.}
}



| @n A marketing team splits their customer base into groups so they can design targeted ad campaigns, based only on data they've collected: like purchasing frequency, average spend, and age.
| Supervised @hspace{3em} @ifsoln-choice{Unsupervised}

@blanklines{3}{
@ifsoln{This is an example of _clustering_, where the computer tries to find "clumps" of unlabeled points in space.}
}



| @n  A real estate app estimates the market price of a house based on features like square footage, number of bedrooms, and zip code, using a model trained on thousands of past home sales with known prices.
| @ifsoln-choice{Supervised} @hspace{3em} Unsupervised

@blanklines{3}{
@ifsoln{The model is trained on a labeled dataset where the sale price is already known. This is the "answer key" that makes it an example of supervised learning}
}



| @n An algorithm analyzes a massive dataset of unlabelled DNA sequences from various plant species to find hidden structural similarities and categorize them into evolutionary clusters.
| Supervised @hspace{3em} @ifsoln-choice{Unsupervised}

@blanklines{3}{
@ifsoln{The algorithm is exploring the data to find underlying patterns and structures on its own, without any target labels or guidance.}
}



| @n A hospital uses a machine learning model to classify medical images (X-rays) as either 'healthy' or 'pneumonia-affected' after training it on a dataset of images reviewed and tagged by expert radiologists.
| @ifsoln-choice{Supervised} @hspace{3em} Unsupervised

@blanklines{3}{
@ifsoln{The model is trained on a labeled dataset where the outcome is already known. This is the "answer key" that makes it an example of supervised learning}

| @n A security system monitors network traffic to detect anomalies. It analyzes normal daily patterns and flags any data packet behavior that deviates significantly from the norm, without using a list of known cyberattack signatures.

@blanklines{3}{
@ifsoln{The algorithm is exploring the data to find underlying patterns and structures on its own, and then search for deviations. There is no labeling or human guidance to define what "normal patterns" are}
