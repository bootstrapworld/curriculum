We've provided a function that will build the optimal decision tree of _any length_, using _any columns_ you choose! Here's the contract:

@vspace{1ex}

@show{(contract "build-tree" '("Table" "List" "String" "Number") "DecisionTree")}

@vspace{1ex}

The `build-tree` function takes *four arguments* in its Domain:

- The _training dataset_ from which to fit the tree
- A list of all the column names to split on
- The name of the _label_ column containing all the "answers" for the various categories
- The _maximum depth_ the tree is allowed to be
