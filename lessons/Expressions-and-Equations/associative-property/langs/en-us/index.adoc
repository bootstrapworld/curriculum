== The Associative Property

=== Overview

Students explore Circles of Evaluation to discover that no matter how we group numbers being either added or multiplied, the end result will always be the same.

=== Launch
@slidebreak{Launch}

We've already learned about the Commutative Property, which tells us that, for any expression involving only addition or only multiplication, changing the order of the numbers will not change the end result. But what about changing the _groupings_?

If we wanted to add 2 + 3 + 5, would it make a difference whether we added 2 + 3 first or started with 3 + 5? 

When we can change the groupings and still get the same answer for a given operation, the operation is @vocab{Associative}.

@QandA{
@Q{Make a prediction: Which operations do you think will be associative?}
}

@lesson-instruction{
- Complete @printable-exercise{discover-assoc.adoc} and @printable-exercise{discover-assoc-cont.adoc} with your partner.
}

@slidebreak{Launch}

@QandA{
@Q{Which operations does the Associative Property apply to?}
}

@slidebreak{Launch}

The Associative Property holds for addition and multiplication, but not division or subtraction.

@lesson-point{
When adding three numbers, the answer will be the same whether we start with the first pair of numbers _or_ the last pair of numbers. +
When multiplying three numbers, the answer will be the same whether we start wiht the first pair of numbers _or_ the last pair of numbers.
}

=== Investigate
@slidebreak{Investigate}

@lesson-instruction{
- Turn to @printable-exercise{true-or-false.adoc}. Determine whether the Associative Property has been applied correctly in each row to yield equivalent expressions.
}

@teacher{
Students will encounter expressions with more than three numeric values and will need to broaden their definition of the Associative Property. The same general rules still apply: When we change the groupings of numbers being added or multiplied, we get the same outcome.
}

@slidebreak{Investigate}

@QandA{
@Q{Can you ever apply the Associative Property on an expression that includes division or subtraction? Explain.}
@A{Yes! While we can't regroup division or subtraction, in a longer expression we might be able to do some regrouping of addition or multiplication, even if the expression also includes subtraction or division.}
}

@slidebreak{Investigate}
One of the main reasons it's useful to be able to rewrite expressions to use different groupings is to simplify computation.

@lesson-instruction{
- Complete @printable-exercise{which-grouping-makes-the-comp-easier.adoc}. Examine each Circle of Evaluation and put a check mark by the one that results in a simpler computation. Then, evaluate the expression.
- Now, try @printable-exercise{associativity-makes-comp-easier.adoc} to apply the Associative Property to expressions with more numbers and varying operators.
- @opt{Looking for more practice?
  * @printable-exercise{assoc-prop-table.adoc} is a chance to draw all possible regroupings using Circles of Evaluation
  * @opt-printable-exercise{associativity-makes-comp-easier-challenge.adoc} focuses on expressions with fractions and decimals
  * @opt-printable-exercise{so-many-groupings.adoc} tackles a very long expression
}
}

== Applying the Commutative and Associative Properties Together

=== Launch
@slidebreak{Launch}

The properties are much more powerful when we understand how to apply them together! 

Consider the expression @big{@math{2 \times 17 \times 5}}. We can...

- *regroup* using just the Associative property @hspace{1em} @math{2 \times (17 \times 5)} 
- *reorder* using just the Commutative Property   @math{(17 \times 2) \times 5} 
- *reorder and regroup* using both properties @hspace{1em} @math{17 \times (2 \times 5)}    which simplifies our mental math to   @math{17 \times 10} 

@slidebreak{Launch}

@lesson-instruction{
- Practice reordering and regrouping addition and multiplication expressions by completing @printable-exercise{restructuring-addition-expressions.adoc} and @printable-exercise{restructuring-multiplication-expressions.adoc}
}

=== Investigate
@slidebreak{Investigate}

The freedom to evaluate expressions in a variety of ways rather than just moving left to right opens up a world of possibility. In the short term, we can compute efficiently. In the long term, confidence using laws of arithmetic provides a strong foundation for making sense of more complex algebraic reasoning.

@lesson-instruction{
- Use your understanding of the Commutative and Associative Properties to complete @printable-exercise{which-not-equivalent.adoc}
- When you're done, see if you can apply your understanding to expressions with variables on @printable-exercise{which-not-equivalent-variables.adoc}
- @opt{Revisit all of the functions and operations from the Commutative and Associative Property lessons by completing @opt-printable-exercise{categorize-functions.adoc}.}
}

@slidebreak{Investigate}

@teacher{With some familiarity of regrouping and reordering, students are ready to get creative and develop their own arithmetic expressions.}

@lesson-instruction{
- Think of an addition or multiplication problem that _appears_ to be very challenging, but is _much_ easier to solve after applying the Associative Property _and_ the Commutative Property. Write it down on a piece of paper. Be creative!
- Trade papers with a partner. How do your problems compare?
- Represent your partner's expression with a Circle of Evaluation that makes solving simpler.
- Turn your paper in to your teacher.
}

@teacher{We encourage you to review students' submissions, and write a few on the board to discuss as a class. We want to sharpen students' eyes and help them develop the ability to spot instances when they might apply the Associative Property in any context.}
