*The Commutative Property is true for any operation where _all orders_ of the numbers produce the same result.*

== Does Order matter for Addition? (or "Is Addition Commutative?")

@n Ever thinks that order doesn't matter for addition. They write the following equations as proof. : @hspace{2em} @big{@math{2 + -3 = -1} @hspace{1em} @math{-3 + 2 = -1}}

@vspace{1ex}

Can you think of any addition problems where order matters? Write at least three more examples in both orders and see if they evaluate to the same thing.

@vspace{5ex}

@n Do you think *addition* is commutative? Explain. @fitb{}{}
