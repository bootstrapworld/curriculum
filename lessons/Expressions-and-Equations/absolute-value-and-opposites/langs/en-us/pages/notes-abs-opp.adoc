= Absolute Value and Opposite
++++
<style>
.circleevalsexp { display: inline; }
</style>
++++

*Opposites* are two numbers that are the same distance from zero on the number line, with one negative and one positive.  

@vspace{1ex}

- Examples:
  * @math{5} is the opposite of @math{-5}
  * @math{-6.1} is the opposite of @math{6.1}
  * any given number represented as @math{h} is the opposite of @math{-h}
- @math{-h}   is read:  “the opposite of h”  or  “negative h”
- @proglang's  `negate` function takes in a number and returns its opposite.

@indented{We can represent  @math{-h}  with a Circle of Evaluation: @show{(coe '(negate h))}}

@vspace{1ex}

*Absolute value* is the (positive) distance of a number from zero. 

@vspace{1ex}

- Examples:
  * @big{@math{|4| = 4}} 
  * @big{@math{|-4| = 4}}
- @big{@math{|h|}}  is read:  "the absolute value of @math{h}"
- Because _opposites_ are the same distance away from zero, they will always have the same absolute value. 
- @proglang's `abs` function takes in a number and returns its absolute value.

@indented{We can represent  @big{@math{|h|}}  with a Circle of Evaluation: @show{(coe '(abs h))}}

@vspace{1ex}

The algebraic expressions   @big{@math{|h|}}  and  @big{@math{-h}}  sometimes produce the same outcome, and they sometimes produce different outcomes: 

@vspace{1ex}

- @big{@math{|h|}}   is always positive or zero
- @big{@math{-h}}   can be negative, zero, or positive.

@vspace{1ex}

We can create expressions that utilize both opposite and absolute value. For instance:

@vspace{1ex}

- We can find the @vocab{opposite} of an @vocab{absolute value}: @math{- | x |}
  * @math{- |2| = -2}
  * @math{- |-2| = -2}
- We can find the @vocab{absolute value} of an @vocab{opposite}. @math{ |- x |}
  * @math{|2| = -2}  
  * @math{|-2| = 2}
  * @math{|- -2| = 2}

@vspace{1ex}

Thinking about the structure of the expression (and studying its Circle of Evaluation) can help us understand if it is positive or negative.
