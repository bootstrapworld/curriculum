Match each function definition to the corresponding table. 

*Note:* The tables are shown sideways to save space.

[.FillVerticalSpace, cols=">.^8a,^.^1a,4,^.^1a,^.^8a", stripes="none", grid="none", frame="none"]

! y ! 1 ! 4 ! 9 ! 16 ! 25
!===
