*Teachers*: To prepare for this activity, cut out the cards below and keep each set (of three) together with a rubber band or paperclip. For this activity, you will also print and cut cards from two other card sort pages: @handout{translating-card-sort-1.adoc, "Arithmetic Expressions"} and @handout{translating-card-sort-3.adoc, "Verbal Expressions"}; we recommend printing each set in a different color. __Note that this page includes three copies of the same set. You will need one set for every pair of students.__
