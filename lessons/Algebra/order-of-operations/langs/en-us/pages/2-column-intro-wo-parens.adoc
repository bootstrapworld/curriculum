= Expressions -> Circles of Evaluation 2

Translate each of the arithmetic expressions below into Circles of Evaluation. 

_@proglang has a dedicated function for squaring a number `sqr`. And a function for exponents `expt`, which takes in both the base and the exponent._
@do{
