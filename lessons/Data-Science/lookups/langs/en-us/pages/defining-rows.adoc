*Remember: rows start at index zero!*

We've already given you two row definitions, `cat-row` and `dog-row`:

```
cat-row = row-n(animals-table,  0)  # Sasha is a cat
dog-row = row-n(animals-table, 10)  # Toggle is a dog
```
